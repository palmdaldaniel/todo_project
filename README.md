# todo_project
A todolist-app made with Vuex

Explored state management, mutations, compononents, props,
eventhandlers, conditionals, Object.create, computed properties,
and redirecting with router.

Features include: 
 - Add todo
 - Remove todo 
 - Update todo
 - Prioritize todo
 - Local storage



## Project setup
```
npm install
```
